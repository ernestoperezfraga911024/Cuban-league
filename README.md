# Cuban League — Primera versión

## Qué incluye
- Los 20 participantes y sus escudos.
- Clasificación adaptable a iPhone, Android y computadora.
- Diseño oscuro profesional.
- Botón para compartir.
- Datos separados en `data.json`.

## Cómo probarla
GitHub Pages publicará estos archivos directamente. Al abrir `index.html` como archivo local, algunos navegadores bloquean la lectura de `data.json`; esto desaparece al publicarla.

## Cómo cambiar puntos provisionalmente
Abre `data.json` y cambia `points` y `played` para cada participante.

## Próximo paso
Conectar la tabla a Google Sheets para actualizarla semanalmente sin tocar archivos.
